# Agentic AI — Learning Notes

A personal, hands-on journey into **agentic AI**, written by a Java developer who is
new to AI and picking up Python along the way.

- **Stack:** Python + **LangChain** (from Phase 2 onward)
- **Approach:** concepts first → raw provider SDK → LangChain abstractions
- **Pace:** ~1 hour/day, phase by phase

!!! tip "Think of it as"
    A lab notebook, not a textbook. Each topic explains one idea in plain language,
    with diagrams and everyday analogies, before any code shows up.

New here? Work through the topics in order — they build on each other.
